# BIDCONF 2026 — сканер регистрации

Статический веб-сканер QR-билетов для входа на конференцию **BIDCONF 2026**.
Работает прямо в браузере телефона (камера), хостится на GitHub Pages — отдельная ссылка,
без сервера.

## Как работает
1. Персонал открывает ссылку сканера, вводит **код доступа**.
2. Наводит камеру на QR билета → сканер показывает гостя: **ФИО, компания, тип билета, № билета**
   и вердикт: ✅ ПРОПУСТИТЬ / ⚠️ УЖЕ ВОШЁЛ / ❌ НЕ НАЙДЕН.
3. Кнопка **«Сканировать следующий →»** — продолжить.
4. Кнопка **«Список гостей»** — кто прошёл / кого ждём, поиск.

## Безопасность данных
Список гостей **зашифрован** (AES-256-GCM, ключ из кода доступа через PBKDF2). Публично лежит
только шифр `tickets.enc` — без кода ФИО не прочитать. Отметки «вошёл» хранятся в браузере
устройства (localStorage).

## Файлы
- `index.html` — сканер (камера, сверка, список гостей)
- `tickets.enc` — зашифрованный реестр (генерится из `bidzaar-tickets/tools/gen_scanner_registry.py`)
- `jsqr.min.js` — QR-декодер (fallback к нативному BarcodeDetector)

## Обновить список гостей
```
cd ~/Projects/bidzaar-tickets && python3 tools/issue_bidconf.py        # реестр билетов
python3 tools/gen_scanner_registry.py "<КОД>"                          # -> bidconf-scanner/tickets.enc
```
Затем закоммитить новый `tickets.enc`.
